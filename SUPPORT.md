# Support

## Getting Help

| Channel                                                                              | Use For                                    |
| ------------------------------------------------------------------------------------ | ------------------------------------------ |
| [GitHub Issues](https://github.com/westbridge-inc/Westbridge-ERP-1/issues)           | Bug reports, feature requests              |
| [GitHub Discussions](https://github.com/westbridge-inc/Westbridge-ERP-1/discussions) | Questions, ideas, general discussion       |
| Email: support@westbridgetoday.com                                                   | Account issues, billing, security concerns |

## Before Opening an Issue

1. Search [existing issues](https://github.com/westbridge-inc/Westbridge-ERP-1/issues) to avoid duplicates
2. Use the appropriate [issue template](https://github.com/westbridge-inc/Westbridge-ERP-1/issues/new/choose)
3. Include your browser version, OS, and steps to reproduce

## Security Vulnerabilities

**Do not** open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
